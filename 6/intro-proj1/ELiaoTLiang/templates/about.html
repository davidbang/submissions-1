What We Did

We used Eric's old analysis project from second term sophomore year. He analyzed rankings of the popular game Osu.
The original project was form based.
Had we used the original format, the user would be able to select certain parts of the table and limit the amount of data visible.
Instead, we used some css and buttons so that the user can select which category to select and which direction the categories should sort in.
We had to convert the original code to a format that satisfied flask commands.
Originally, the project involved printing several lines of html markup, especially for the table.
Now, we converted the function so that it creates a string of html markup that would be inserted into a template.
