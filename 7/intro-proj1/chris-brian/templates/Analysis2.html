The Analysis

Background

Looking at this data and using our knowledge of League of Legends, we soon came up with two ideas for trends in the data. First, we wondered if champions were priced based upon their viability and thus have a better win rate. If this were true, that means that League of Legends is not a fair game and you can "buy" skill in this game. If not, then the game would be fair. After we found that out, we moved on to a concept in League: ranged champions can attack at a safer distance that melee champions, therefore taking less damage than them. We now wanted to see if Riot gave an advantage to melee champions to counteract this disadvantage. Thus, we looked at base armor and decided to calculate the average base armor of melee champs with the average base armor of ranged ones.

Analysis

 1) Looking at this data, we find that there are 5 IP costs(450,1350,3150,4800,6300) Using a code, we can find the average win rate of each tier of IP costs. Thus, we have:

IP Cost Win Rate
450 0.49077
1350 0.501472727273
3150 0.480554166667
4800 0.497158333333
6300 0.492684375


The win rate of all 5 tiers of IP costs are all very near 50%, thus we can conclude that IP cost does not determine how good a champion is.

2) We also used a code to find the average starting armor value for ranged champions compared to melee champions:

Range Armor
Ranged 11.7540816327
Melee 16.4103174603


According to the data, we found that Riot has compensated for melee champions by increasing their starting armor! This allows them to go up against ranged champions without being at a huge disadvantage by taking too much damage.

Conclusion

 According to the above data, we can conclude that: League of Legends is definitely a fair game. The fact that the champions with the second to least amount of IP Cost has the highest winning rate shows that skill cannot be bought in this game.
