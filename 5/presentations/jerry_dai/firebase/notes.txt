Firebase
- an API that allows for realtime updates and data sync
- data is stored as JSON blocks into a Firebase "database"
- "snapshots" are used to return changes to the data back to the webpage,
or whatever platform is being used
- basic Firebase is free, pay for more storage and more connections to the Firebase
